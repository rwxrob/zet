# Booting into BIOS on any system

	•	Dell: F2 or F12
	•	HP: F10 or Esc
	•	Lenovo: F1, F2, or Enter
	•	Acer: F2 or Del
	•	Asus: Del or F2
	•	MSI: Del or F2
	•	Gigabyte: Del or F2
	•	Toshiba: F2 or Esc
	•	Samsung: F2
	•	Sony: F2 or Assist button (on some models)
	•	Microsoft Surface: Hold Volume Up button while powering on.
